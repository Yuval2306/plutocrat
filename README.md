# 💰 PLUTOCRAT

> **A luxury-themed billionaire spending simulator.**  
> Pick one of the world’s richest people and try to spend their fortune on yachts, rockets, private islands, sports teams, skyscrapers, and more.

🌐 **Live Demo:** (https://yuval2306.github.io/plutocrat/)

---

## ✨ Features

- **💸 Interactive Spending Simulator** — Choose a billionaire and spend their net worth in real time
- **🛥️ Luxury Purchase Catalog** — Buy yachts, private jets, rockets, islands, sports teams, and more
- **📊 Live Stats Dashboard** — Track total spent, remaining balance, percentage spent, and items purchased
- **🔎 Search & Filter** — Search billionaires and browse purchases by category
- **🌍 Bilingual Support** — Switch between English and Hebrew
- **📱 Responsive Design** — Optimized for desktop and mobile screens
- **🎨 Premium UI/UX** — Dark luxury design with gold accents and smooth interactions
- **⚡ Vanilla Frontend** — Built with pure HTML, CSS, and JavaScript

## 🎮 How It Works

1. Select a billionaire from the top 100 list  
2. Browse luxury items by category  
3. Click items to “buy” them  
4. Watch the stats update instantly  
5. Switch between English and Hebrew anytime

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Yuval2306/Plutocrat.git
cd Plutocrat
```

### 2. Open the project

Because this is a static HTML/CSS/JavaScript project, you can simply open:

```bash
index.html
```

Or use a local development server:

```bash
npx serve .
```
